    • and(firstOperand: DoubleWord, secondOperand: DoubleWord): DoubleWord

      This method performs an bit-wise, logical AND operation on two given binary values, according to the following table.

      Bit x Bit y Result
      0 0 0
      1 0 0
      0 1 0
      1 1 1
