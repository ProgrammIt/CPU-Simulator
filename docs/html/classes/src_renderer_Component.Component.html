    Class ComponentAbstract

    This class represents a generic component of the graphical user interface. A component is a bundle of a TypeScript class and an accompanied HTML template file. It provides an interface for (re-)render the components with the members values of the TypeScript class. Rendering means the creation of a DocumentFragment, which can be inserted into the document object model (DOM).
